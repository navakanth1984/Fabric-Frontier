# Release v1.8

## Overview
Transitions ACDLC into a full Agent Operating System Platform.

## Features
- **Persistent Identity Store**: Replaced in-memory dictionaries with a `StorageProvider` interface backed by JSONL. Profiles, trust, and reputation are now durable across restarts.
- **Multi-Tenant Isolation**: Added tenant registries, shared token quotas, and strict pre-scoring isolation.
- **Distributed Worker Nodes**: Established `cluster/` directory with `node_registry.py`, `node_health.py`, and `node_scheduler.py` for multi-process worker pools.
- **Autonomous Learning Engine**: Added `routing_optimizer.py`, `trust_calibrator.py`, and `policy_recommender.py` to close the routing feedback loop by auto-generating recommended weights based on execution outcomes.
- **Model Registry**: Added `models/` directory for routing to foundational models (GPT, Claude, Gemini) based on capabilities and cost limits.
