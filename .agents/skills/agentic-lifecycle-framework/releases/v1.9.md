# Release v1.9

## Overview
Evolves the ACDLC Platform into a true **Distributed Agent Platform**. This release shifts focus from single-node agent execution logic to operational resilience, distributed systems coordination, and strict governance safety.

## Features
- **Durable Event Store & Execution Replay Engine**: The `EventBus` now sinks into a persistent `EventStore` append-only log, enabling full time-travel debugging via `ReplayEngine`.
- **Leader Election & Split Brain Prevention**: Lightweight cluster consensus algorithm guaranteeing a single orchestrating leader, augmented with a `TermStore`.
- **Agent Reputation Persistence**: Reputation state is now purely event-sourced (success, failure, violation) allowing complete reconstruction.
- **Policy Promotion Workflow**: Governance safety via `PolicyPromoter`. Recommended weights move through `staging/` and `approved/` gates before touching `production/`.
- **Cluster Autoscaler**: Elastic scaling decisions triggered by analyzing task queue depth against the healthy/degraded node capacities.
