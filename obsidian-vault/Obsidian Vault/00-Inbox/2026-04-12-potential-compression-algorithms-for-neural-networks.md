# Potential Compression Algorithms for Neural Networks

## Key Idea
This note aims to explore existing literature on compression algorithms for neural networks and identify potential candidates for implementation.

## Details
Compression algorithms have the potential to significantly impact the efficiency of deep learning models. Literature suggests several promising techniques, including:
- Huffman coding: A method for lossless data compression that can be adapted to compressed neural network weights.
- Arithmetic Coding: A variable-length prefix code that can be used to compress neural network activations.
- Quantized Neural Networks (QNNs): Techniques that reduce the precision of model parameters and activations while maintaining performance.
